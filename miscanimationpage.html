Miscellaneous Animations

This collection features various motion graphics pieces I created using After Effects.

I created 2 animations for social media platforms for a local drive-in theater to promote upcoming movies. For these projects, I created vector art assets in Illustrator, then animated and rigged them in AfterEffects. For the other animations, I drew character illustrations in Procreate, then brought those elements into AfterEffects for motion tweening practice.
